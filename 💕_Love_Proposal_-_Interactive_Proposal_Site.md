# 💕 Love Proposal - Interactive Proposal Site

Um site interativo e romântico para fazer um pedido de namoro especial!

## 🎯 Características

✨ **Página Inicial**
- Efeito typewriter com a pergunta "Quer namorar comigo?"
- Botão "Sim!" que leva para a página de confirmação
- Botão "Não" que foge do cursor
- Emojis de coração que explodem ao clicar na tela

💕 **Página de Confirmação**
- Corações subindo pela tela continuamente
- Pétalas caindo suavemente
- Pop-up estilo janela de desktop com a mensagem "eu te amo, CABEÇUDA!!"
- GIF animado do personagem feliz
- Botão de mudo para controlar a música

## 🚀 Como Usar

### Opção 1: Abrir Direto
Basta abrir o arquivo `index.html` no seu navegador!

### Opção 2: Usar um Servidor Local
```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (http-server)
npx http-server

# Com PHP
php -S localhost:8000
```

Depois acesse `http://localhost:8000` no seu navegador.

## 📁 Arquivos Inclusos

- `index.html` - Arquivo HTML principal (já compilado)
- `assets/` - Pasta com CSS e JavaScript compilados
- `.gitignore` - Arquivo para ignorar pastas desnecessárias no Git

## 🎨 Personalização

Se você quiser modificar o site, você precisará do código-fonte React. Consulte o repositório principal em GitHub.

### Mudar a mensagem do pop-up
1. Edite o arquivo `index.html`
2. Procure por "eu te amo, CABEÇUDA!!"
3. Substitua pelo seu texto

## 🌐 Deploy no GitHub Pages

1. Crie um repositório no GitHub chamado `love_proposal`
2. Clone o repositório localmente
3. Copie todos os arquivos desta pasta para o repositório
4. Execute:
   ```bash
   git add .
   git commit -m "Initial commit: Love proposal site"
   git push origin main
   ```
5. Vá para as configurações do repositório
6. Em "Pages", selecione "Deploy from a branch"
7. Escolha a branch `main` e a pasta `/ (root)`
8. Clique em "Save"

Seu site estará disponível em: `https://seu-usuario.github.io/love_proposal`
